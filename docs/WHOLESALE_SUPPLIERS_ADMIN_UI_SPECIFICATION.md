- All tabs visible

### Tablet (768px-1023px)

- Tabs as dropdown on smaller screens

## Permissions

Different admin roles:

### Super Admin
- All actions
- Change settings
- Delete suppliers
- Manage blacklist

### Moderator
- Approve/reject suppliers
- Merge duplicates
- Edit supplier data
- View logs

### Viewer
- View suppliers
- View logs
- Export data
- No modifications
